# Cirrus migration 2025

!!! important
    This information was last updated on 20 Nov 2025

This section of the documentation is a guide on the migration process from the 
old Cirrus system (SGI ICE XA) to the new Cirrus system (HPE Cray EX4000)
in Q4 2025.

!!! important
    The date for the switch from old Cirrus system to the new Cirrus
    system is Monday 8 December 2025.

    After the switch you will no longer be able to access the old Cirrus login
    and compute nodes or the RPOOL (/scratch) solid state storage.

More information is available on the following pages

   - [Information for users and projects not continuing on Cirrus EX4000](not-migrating.md), includes:
      + Requirement to transfer data off Cirrus before switch date
      + Information on how to apply for access to Cirrus EX4000
   - [Migrating your account to Cirrus EX4000](account-migration.md), includes:
      + Checking if your project/account will migrate to Cirrus EX4000
      + Checking how much resource you will have on Cirrus EX4000
      + How to setup your account on Cirrus EX4000
   - [Migrating data to Cirrus EX4000](data-migration.md), includes:
      + How existing project/user data will be migrated to Cirrus EX4000
      + Requirement to transfer data off /scratch (RPOOL) before switch date
   - [Main differences between old Cirrus and Cirrus EX4000](cirrus-differences.md), includes:
      + Summary of hardware differences
      + Summary of software differences


!!! tip
    If you need help or have questions on this migration process, 
    please [contact the Cirrus service desk](https://www.cirrus.ac.uk/support-access/user-support/)
