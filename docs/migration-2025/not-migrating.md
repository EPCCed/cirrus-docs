# Impacts for projects not migrating to Cirrus EX4000

!!! important
    This information was last updated on 20 Nov 2025

This page summarises the impacts on users and projects who are not
migrating to Cirrus EX4000 on 8 December 2025. It also highlights any
actions you need to take.

!!! tip
    If you are unsure whether or not your project is migrating to 
    Cirrus EX4000, see the information on the
    [account migration page](https://docs.cirrus.ac.uk/migration-2025/account-migration/).

## Data access

Projects and user accounts that are not migrating to Cirrus EX4000
will lose access to any data on Cirrus from *08:00 GMT on Monday 8 December 2025*.

**Users and projects should copy any data they wish to keep off Cirrus before 
08:00 on Monday 8 December 2025.**

## Getting access to Cirrus EX4000

If projects want to access Cirrus EX4000 for their research, there are a number
of different access routes. Please consult the
[Access page on the Cirrus website](https://www.cirrus.ac.uk/support-access/access/)
for details on how to do this.
